Interview cheat sheet:
  execute()     → void, exceptions swallowed to UncaughtExceptionHandler
  submit()      → Future, exception wrapped in ExecutionException on get()
  shutdown()    → orderly stop, queue drains
  shutdownNow() → interrupt workers, return pending tasks
  thenApply     → map value
  thenCompose   → chain async step (flatMap)
  tryLock       → don't block forever on contested lock
